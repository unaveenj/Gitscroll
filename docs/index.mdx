GitScroll is a developer discovery tool that turns the GitHub Search API into an infinite, swipeable flash-card deck. Instead of browsing lists, you swipe through curated repositories one at a time, save the ones that spark ideas, and move on.

## What GitScroll does

GitScroll fetches repositories from the GitHub Search API that have between **30 and 500 stars** and were pushed to after 2024-01-01 — the hidden-gem range. Each repo is categorised as **Trending**, **AI Tool**, **Dev Tool**, or **Fun Project**, then surfaced one card at a time.

```
GitHub Search API → categorisation → feed interleaving → card stack → swipe
```

Key behaviours:

- **Random starting page** — every session begins on a different page of results so you never see the same repos twice.
- **Infinite pagination** — the next page is pre-fetched silently when you are four cards from the end.
- **Local favourites** — saved repos are persisted to `localStorage` with no backend required.
- **Surprise Me** — a button that jumps to a random card in the current deck.
