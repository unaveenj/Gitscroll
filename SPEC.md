# SPEC.md

## Product Vision

GitScroll helps developers discover GitHub projects through an addictive scrolling interface.

## Core User Flow

User opens app
↓
First repo card appears
↓
User scrolls vertically
↓
User swipes right to save
↓
User swipes left to discard

## Card Content

Repo Name
Description
Tech stack
Stars
Idea suggestion
Buttons

Example:

🔥 AI Voice Cloner

Clone voices using AI with 3 seconds of audio.

Tech: Python, PyTorch

⭐ 18k stars

💡 Idea: Build a personalised podcast generator

[Star] [Save] [Open Repo]
